# 5과목 06 — 정보보안 윤리

> 출제기준 매핑: 2.1 정보보안 윤리 — 사이버 윤리 (보안윤리 개념·디지털 저작권 침해 및 보호기술·유해정보유통·사이버 폭력·사이버 사기 등 범죄행위) / 정보시스템 이용자 및 개인정보취급자의 금지행위

## §0. 들어가며

본 단원은 정보화 사회의 윤리적 규범과 그 법적 강제력을 다룬다. 정보보안 윤리는 단순한 도덕적 권고가 아니라 한국 법령상 **정보시스템 이용자의 금지행위 ([정보통신망법] §48)** 와 **개인정보 취급자의 금지행위 ([개인정보보호법] §59)** 로 강제되며, 위반 시 형사 처벌이 따른다. 본 단원은 정보보호 윤리의 개념·디지털 저작권·유해정보 유통·사이버 폭력·사이버 사기 영역을 다룬 후, 정보시스템 이용자 및 개인정보 취급자의 금지행위를 법령 조문 단위로 정리한다.

본 단원의 1차 출처는 모두 국가법령정보센터의 현행본 (2026.04.30 기준) 이다. [정보통신망법] [^infonet-act], [개인정보보호법] [^pipa], [정보통신기반 보호법] [^cii-act], [저작권법] [^copyright-act], [형법] [^crim-act], [스토킹범죄의 처벌 등에 관한 법률] [^stalking-act], [성폭력범죄의 처벌 등에 관한 특례법] [^sexual-act], ACM Code of Ethics (2018) [^acm-ethics], IEEE Code of Ethics [^ieee-ethics] 를 사용한다.

학습 순서:
1. 정보보안 윤리의 개념과 직업윤리 (§1)
2. 디지털 저작권과 보호기술 (§2)
3. 유해정보 유통 (§3)
4. 사이버 폭력·사이버 사기 (§4)
5. 정보시스템 이용자의 금지행위 — [정통망법] §48 (§5)
6. 개인정보 취급자의 금지행위 — [개보법] §59 (§6)

## §1. 정보보안 윤리의 개념

> **정보보안 윤리 (Information Security Ethics)** : 정보 사회에서 정보를 다루는 모든 주체가 따라야 할 도덕적·법적 규범의 총체. 정보 자산의 적절한 사용·보호·신뢰 구축을 위한 행동 원칙.

### §1-1. 정보윤리의 4대 가치

리처드 메이슨 (Richard Mason) 의 1986년 논문 "Four Ethical Issues of the Information Age" [^mason-pap] 에서 제시된 4대 가치 (PAPA) 는 정보윤리 논의의 출발점으로 인용된다.

| 가치 | 영문 | 핵심 질문 |
|---|---|---|
| 사생활 (Privacy) | Privacy | 개인의 어떤 정보를 누구에게, 어떤 조건으로 공개할 것인가 |
| 정확성 (Accuracy) | Accuracy | 정보의 정확성을 누가 책임지는가 |
| 재산권 (Property) | Property | 정보의 소유권은 누구에게 있는가 |
| 접근권 (Accessibility) | Accessibility | 누가 어떤 정보에 어떤 조건으로 접근할 수 있는가 |

### §1-2. ACM Code of Ethics (2018)

ACM (Association for Computing Machinery) Code of Ethics and Professional Conduct (2018) [^acm-ethics] 의 4개 섹션:

| 섹션 | 내용 |
|---|---|
| 1. 일반 윤리 원칙 (General Ethical Principles) | 사회와 인류의 복지 기여, 해를 끼치지 않을 것, 정직, 공정, 사생활 존중, 기밀 유지, 등 |
| 2. 전문가의 책임 (Professional Responsibilities) | 자신의 역량 안에서 일할 것, 직업적 기준 준수, 시스템에 대한 책임, 등 |
| 3. 전문가 리더십 (Professional Leadership Principles) | 사회적 책임, 윤리·법규·기준 준수, 등 |
| 4. 준수 (Compliance with the Code) | 코드 위반 시 책임 |

### §1-3. IEEE Code of Ethics

IEEE Code of Ethics [^ieee-ethics] 의 핵심 약속:

- 안전·건강·복지·환경에 부합하는 결정
- 이해 충돌 회피 / 발견 시 공개
- 정직성·현실성에 기반한 진술
- 부패·뇌물 거부
- 기술과 그 사회적 영향에 대한 이해 증진
- 자신과 타인의 역량 개선
- 정직한 비판과 오류 수용·수정
- 공정한 처우 (인종·종교·성별 등 차별 금지)
- 안전·재산·환경·평판에 대한 위해 회피
- 동료 지원

### §1-4. 시험 포인트

- 정보윤리 4대 가치 (PAPA) = **Privacy·Accuracy·Property·Accessibility**.
- ACM Code of Ethics 발행 = **2018년**.
- 정보보안 직업윤리는 **권고** 가 아닌 **법적 강제** 영역과 **자율적 윤리** 영역이 공존한다.

## §2. 디지털 저작권과 보호기술

### §2-1. 저작권법의 보호 대상

[저작권법] §4 [^copyright-act] 는 다음을 저작물로 예시한다.

| 카테고리 | 사례 |
|---|---|
| 어문저작물 | 소설·시·논문·강연 |
| 음악저작물 | 작곡·가사 |
| 연극저작물 | 무용·무언극 |
| 미술저작물 | 회화·서예·조각·공예·응용미술 |
| 건축저작물 | 건축물·건축모형·설계도서 |
| 사진저작물 | 사진 |
| 영상저작물 | 영화·드라마 |
| 도형저작물 | 지도·도표·설계도·약도·모형 |
| 컴퓨터프로그램저작물 | 소프트웨어 |

### §2-2. 디지털 저작권 침해 유형

| 침해 유형 | 사례 |
|---|---|
| 복제권 침해 | 무단 다운로드·CD 굽기·서버 업로드 |
| 공중송신권 침해 | 무단 스트리밍·웹 게시·P2P 공유 |
| 2차 저작물 작성권 침해 | 무단 번역·각색·재편집 |
| 동일성 유지권 침해 | 무단 변형·왜곡 |
| 복제 방지·보호기술 회피 | 기술적 보호조치 무력화 |

### §2-3. 기술적 보호조치 (TPM)

> **기술적 보호조치 (Technological Protection Measures, TPM)** : 저작물에 대한 접근을 효과적으로 통제하거나 저작권 등의 침해 행위를 효과적으로 방지·억제하기 위한 기술적 조치. [저작권법] §2 정의 28호 [^copyright-act].

[저작권법] §104-2 [^copyright-act] 는 다음 두 가지 행위를 금지한다.

| 행위 | 내용 |
|---|---|
| 보호조치 무력화 | 정당한 권한 없이 보호조치를 제거·변경·우회하는 것 |
| 무력화 도구 제공 | 무력화 도구를 제조·수입·공중에게 양도·대여·전송·전시·제공·청약하거나 광고하는 것 |

위반 시 [저작권법] §136 [^copyright-act] 에 따라 형사 처벌 (3년 이하 징역 또는 3천만원 이하 벌금) 가능.

### §2-4. DRM — 디지털 저작권 관리

> **DRM (Digital Rights Management)** : 디지털 콘텐츠의 사용·복제·배포를 제어하기 위한 기술적·법적 메커니즘.

DRM 의 핵심 요소:

| 요소 | 기능 |
|---|---|
| 콘텐츠 식별 | 콘텐츠 ID·메타데이터·워터마킹 |
| 암호화 | 콘텐츠 자체 암호화 |
| 라이선스 관리 | 사용 권한 표현 (REL) 과 강제 |
| 사용자 인증 | 콘텐츠 접근 시 사용자 검증 |
| 사용 추적 | 사용 내역 모니터링·과금 |

### §2-5. 디지털 워터마킹·핑거프린팅

| 기술 | 정의 | 목적 |
|---|---|---|
| 디지털 워터마킹 | 콘텐츠에 **저작권자 정보** 를 삽입 | 저작권 식별·소유권 증명 |
| 디지털 핑거프린팅 (포렌식 워터마킹) | 콘텐츠에 **사용자별 고유 식별자** 를 삽입 | 불법 유통 시 유출자 추적 |
| 스테가노그래피 | 정보 자체를 미디어 안에 숨김 (보안 목적) | 비밀 통신 / 안티 포렌식 (5과목 04 §9 cross-ref) |

### §2-6. 시험 포인트

- 컴퓨터 프로그램도 **저작권법의 저작물** ([저작권법] §4).
- 기술적 보호조치 무력화 도구 제공 = [저작권법] §104-2 위반.
- 디지털 워터마킹 = **저작권자** 식별 / 핑거프린팅 = **사용자** 식별.
- DRM 의 핵심 = 콘텐츠 암호화 + 라이선스 관리.

## §3. 유해정보 유통

### §3-1. 정통망법 §44 — 권리침해정보의 유통 금지

[정보통신망 이용촉진 및 정보보호 등에 관한 법률] §44 [^infonet-act] 는 정보통신망을 통한 **타인의 권리를 침해하는 정보** 의 유통을 금지한다.

- §44 ①항: 이용자는 사생활 침해·명예훼손 등 타인의 권리를 침해하는 정보를 정보통신망을 통하여 유통시켜서는 아니 된다.
- §44 ②항: 정보통신서비스 제공자는 자신이 운영·관리하는 정보통신망에 위 정보가 유통되지 아니하도록 노력하여야 한다.

### §3-2. 정통망법 §44-7 — 불법정보의 유통금지

[정보통신망법] §44-7 [^infonet-act] 은 다음 정보의 유통을 금지한다.

| 호 | 금지 정보 |
|---|---|
| 1호 | 음란한 부호·문언·음향·화상 또는 영상을 배포·판매·임대하거나 공공연하게 전시하는 내용의 정보 |
| 2호 | 사람을 비방할 목적으로 공공연하게 사실이나 거짓의 사실을 드러내어 타인의 명예를 훼손하는 내용의 정보 |
| 3호 | 공포심·불안감을 유발하는 부호·문언·음향·화상 또는 영상을 반복적으로 상대방에게 도달하도록 하는 내용의 정보 |
| 4호 | 정당한 사유 없이 정보통신시스템·데이터·프로그램을 훼손·멸실·변경·위조·운용을 방해하는 내용의 정보 |
| 5호 | 청소년보호법에 따른 청소년유해매체물 |
| 6호 | 사행행위 등 규제 및 처벌 특례법에 따른 사행행위에 해당하는 정보 |
| 6의2~9호 | 개인정보 거래·국가기밀 누설·국가보안법 위반 등 국가법령상 금지된 정보 |

### §3-3. 정통망법 §44-7 위반 시 처벌

| 행위 | 벌칙 |
|---|---|
| §44-7 ①항 1호 (음란물) 위반 | 1년 이하 징역 또는 1천만원 이하 벌금 ([정통망법] §74) |
| §44-7 ①항 2호 (명예훼손) 위반 | 사실 적시: 3년 이하 징역 또는 3천만원 이하 벌금. 거짓 사실 적시: 7년 이하 징역, 10년 이하 자격정지 또는 5천만원 이하 벌금 ([정통망법] §70) |
| §44-7 ①항 3호 (반복적 공포·불안 유발) 위반 | 1년 이하 징역 또는 1천만원 이하 벌금 ([정통망법] §74) |

> *주의*: 위 벌칙 수치는 [정통망법] 현행본 기준 (2026.04.30) 이며, 시험 응시 시점에 국가법령정보센터에서 직접 확인 필요. 특히 명예훼손 형량 (§70) 은 정기적으로 강화 개정되는 영역이다.

### §3-4. 청소년보호법

[청소년보호법] [^youth-act] 은 청소년의 인격적·신체적 발달을 저해하는 매체물·약물·환경으로부터 청소년을 보호하기 위한 법률이다.

| 분류 | 내용 |
|---|---|
| 청소년유해매체물 | 청소년보호위원회가 결정·고시한 매체물 (음란·폭력·사행성 등) |
| 청소년 유해환경 | 청소년 출입금지업소 (유흥주점·노래연습장 등) |
| 청소년 보호 의무자 | 매체물 제공자·청소년 시설 운영자·교육자 |

### §3-5. 시험 포인트

- 정보 유통 금지 핵심 조문 = **[정통망법] §44-7 ①항 1~9호**.
- 명예훼손 (§44-7 ①항 2호) 처벌 = [정통망법] **§70** (사실 적시 / 거짓 사실 적시 차등).
- 청소년 유해매체물 = [청소년보호법] 정의.

## §4. 사이버 폭력·사이버 사기

### §4-1. 사이버 명예훼손

| 죄명 | 법령 | 형량 |
|---|---|---|
| 형법상 명예훼손 | [형법] §307 [^crim-act] | 사실 적시 2년 이하 징역·금고 또는 500만원 이하 벌금 / 허위 사실 적시 5년 이하 징역, 10년 이하 자격정지 또는 1천만원 이하 벌금 |
| 형법상 모욕 | [형법] §311 | 1년 이하 징역·금고 또는 200만원 이하 벌금 |
| 정통망법상 명예훼손 | [정통망법] §70 [^infonet-act] | 사실 적시 3년 이하 / 거짓 사실 7년 이하 (형법보다 가중) |

> 정통망법 §70 의 명예훼손 처벌은 정보통신망 (인터넷) 을 이용한 경우에 적용되며 형법 §307 보다 가중 처벌된다.

### §4-2. 사이버 스토킹

[스토킹범죄의 처벌 등에 관한 법률] (스토킹처벌법) [^stalking-act] 은 2021년 4월 20일 제정·공포되어 2021년 10월 21일부터 시행되었다. 사이버 공간에서의 반복적 접근·메시지 전송 등도 스토킹 행위에 해당한다.

| 항목 | 내용 |
|---|---|
| 시행일 | 2021.10.21 |
| 스토킹 행위 정의 | 상대방의 의사에 반하여 정당한 이유 없이 상대방 또는 그의 동거인·가족에 대하여 다음 행위를 하여 상대방에게 불안감 또는 공포심을 일으키는 것 |
| 사이버 스토킹 행위 | 정보통신망을 이용하여 물건·글·말·부호·음향·그림·영상·화상을 도달하게 하는 행위 |
| 처벌 | 스토킹범죄 = 3년 이하 징역 또는 3천만원 이하 벌금. 흉기 등 휴대 시 5년 이하 또는 5천만원 이하 |

### §4-3. 디지털 성폭력

| 죄명 | 법령 |
|---|---|
| 카메라등이용촬영죄 | [성폭력범죄의 처벌 등에 관한 특례법] §14 [^sexual-act] (불법 촬영물 제작·유포) |
| 허위영상물 등의 반포 | [성폭력처벌법] §14-2 (딥페이크 등 합성 영상물의 제작·유포) |
| 통신매체이용음란 | [성폭력처벌법] §13 |

### §4-4. 사이버 사기

| 유형 | 설명 | 적용 법령 |
|---|---|---|
| 보이스 피싱 (Vishing) | 전화로 정보 탈취·송금 유도 | 형법 §347 (사기), [전기통신금융사기 피해 방지 및 피해금 환급에 관한 특별법] [^vishing-act] |
| 스미싱 (Smishing) | SMS·MMS 로 악성 URL·정보 탈취 | 형법 §347, 정통망법 §48 (정보통신망 침해) |
| 메모리 해킹 | 인터넷뱅킹의 메모리 변조로 송금 정보 탈취 | 형법 §347 |
| 파밍 (Pharming) | DNS·hosts 변조로 가짜 사이트 유도 | 형법 §347, 정통망법 §48 |
| 몸캠 피싱 | 영상통화로 음란 영상 촬영 후 유포 협박 | 형법 §347, 성폭력처벌법 §14 |

### §4-5. 시험 포인트

- 정통망법 §70 (명예훼손) 은 형법 §307 (명예훼손) 보다 **가중 처벌**.
- 스토킹처벌법 시행 = **2021.10.21**.
- 카메라등이용촬영죄 = [성폭력처벌법] **§14** (성폭력처벌특례법).
- 딥페이크 등 합성 영상 = [성폭력처벌법] **§14-2** (허위영상물 등의 반포).

## §5. 정보시스템 이용자의 금지행위 — [정통망법] §48

> **[정보통신망법] §48 (정보통신망 침해행위 등의 금지)** [^infonet-act] : 정보통신망의 안전성·신뢰성을 보호하기 위해 정보시스템 이용자가 절대 해서는 안 되는 행위를 정의한 핵심 조문.

### §5-1. §48 ①항 — 무단 침입 금지

> "누구든지 **정당한 접근권한 없이** 또는 **허용된 접근권한을 초과** 하여 정보통신망에 침입하여서는 아니 된다."

- 해킹·부정 접근의 가장 핵심적 금지 조문.
- "권한 없이" + "권한을 초과" 두 형태 모두 금지.
- 처벌: [정통망법] §72 ①항 1호 — **3년 이하 징역 또는 3천만원 이하 벌금**.

### §5-2. §48 ②항 — 악성프로그램 금지

> "누구든지 정당한 사유 없이 정보통신시스템·데이터·프로그램 등을 훼손·멸실·변경·위조하거나 그 운용을 방해할 수 있는 프로그램 (이하 '악성프로그램') 을 **전달·유포** 하여서는 아니 된다."

- 악성코드 (바이러스·웜·트로이목마·랜섬웨어·백도어) 전달·유포 금지.
- 처벌: [정통망법] §70-2 — **7년 이하 징역 또는 7천만원 이하 벌금** (단순 침입 §48 ①항보다 가중 처벌).

### §5-3. §48 ③항 — 대량 신호 등의 송신 (DoS) 금지

> "누구든지 정보통신망의 안정적 운영을 방해할 목적으로 **대량의 신호 또는 데이터** 를 보내거나 **부정한 명령** 을 처리하도록 하는 등의 방법으로 정보통신망에 장애가 발생하게 하여서는 아니 된다."

- DoS·DDoS 공격, 부정 명령 (SQL Injection, Command Injection 등) 금지.
- 처벌: [정통망법] §71 ①항 10호 — **5년 이하 징역 또는 5천만원 이하 벌금**.

### §5-4. 정통망법 §48 의 3 가지 금지행위 요약

| 항 | 금지행위 | 처벌 조문 | 형량 |
|---|---|---|---|
| ①항 | 무단 침입 (해킹) | §72 ①항 1호 | 3년 이하 징역 / 3천만원 이하 벌금 |
| ②항 | 악성프로그램 전달·유포 | §70-2 | 7년 이하 징역 / 7천만원 이하 벌금 |
| ③항 | DoS 등 정보통신망 장애 유발 | §71 ①항 10호 | 5년 이하 징역 / 5천만원 이하 벌금 |

> *주의*: 위 형량은 작성 시점 (2026.04.30) 의 [정통망법] 현행본 기준. 정기 개정으로 형량 변경 가능 — 시험 응시 시점 국가법령정보센터 직접 확인 필요.

### §5-5. [정보통신기반 보호법] §12 — 침해 금지

[정보통신기반 보호법] §12 [^cii-act] 는 주요정보통신기반시설에 대한 다음 행위를 금지한다.

| 행위 | 처벌 |
|---|---|
| 1호. 접근권한을 가지지 아니하는 자의 침입 | [기반보호법] §28 — 10년 이하 징역 또는 1억원 이하 벌금 |
| 2호. 데이터 무단 변경·삭제 | 동일 |
| 3호. 운용 방해 | 동일 |

기반보호법 §12 의 처벌은 [정통망법] §48 보다 **가중 처벌** (10년 이하) 된다. 이는 주요정보통신기반시설이 국가 안보·경제에 미치는 영향이 크기 때문이다.

### §5-6. 시험 포인트

- [정통망법] §48 = 정보시스템 이용자의 **3대 금지행위** (해킹·악성코드·DoS).
- 악성프로그램 유포 = §48 ②항 / 처벌 = §70-2 (7년 이하).
- 무단 침입 = §48 ①항 / 처벌 = §72 (3년 이하).
- 주요정보통신기반시설 침해 = [기반보호법] §12 / §28 — 10년 이하 (가중).

## §6. 개인정보 취급자의 금지행위 — [개보법] §59

> **[개인정보보호법] §59 (금지행위)** [^pipa] : **개인정보를 처리하거나 처리하였던 자** 는 다음 각 호의 어느 하나에 해당하는 행위를 하여서는 아니 된다.

### §6-1. §59 의 3대 금지행위

| 호 | 금지행위 |
|---|---|
| **1호** | 거짓이나 그 밖의 부정한 수단·방법으로 개인정보를 **취득** 하거나 **처리에 관한 동의** 를 받는 행위 |
| **2호** | 업무상 알게 된 개인정보를 **누설** 하거나 **권한 없이 다른 사람이 이용** 하도록 제공하는 행위 |
| **3호** | 정당한 권한 없이 또는 허용된 권한을 초과하여 다른 사람의 개인정보를 **훼손·멸실·변경·위조 또는 유출** 하는 행위 |

### §6-2. §59 위반 시 처벌

[개인정보보호법] §71 [^pipa] 의 벌칙 (작성 시점 현행본 기준):

| 위반 유형 | 처벌 |
|---|---|
| §59 1호 위반 (거짓 취득·동의) | 5년 이하 징역 또는 5천만원 이하 벌금 |
| §59 2호 위반 (누설·무단 이용) | 5년 이하 징역 또는 5천만원 이하 벌금 |
| §59 3호 위반 (훼손·변경·유출) | 5년 이하 징역 또는 5천만원 이하 벌금 |

> *주의*: 위 형량은 [개인정보보호법] §71 의 작성 시점 현행본 기준이며, 정기 개정으로 변경될 수 있다. 시험 응시 시점에 국가법령정보센터에서 직접 확인 필요.

### §6-3. §59 의 적용 대상

§59 의 "개인정보를 처리하거나 처리하였던 자" 는 다음을 포괄한다.

| 적용 대상 | 사례 |
|---|---|
| 개인정보처리자 | 개인정보를 처리하는 공공기관·법인·단체·개인 |
| 개인정보 취급자 | 처리자의 지휘·감독 하에 개인정보를 처리하는 임직원·파견근로자·시간제근로자 등 |
| 수탁자 | 개인정보 처리업무를 위탁받은 자 |
| **퇴직자** | "처리하였던 자" — 퇴직 후에도 적용 |

### §6-4. §59 vs §17·§18 차이

§59 는 개인정보의 **부정한 취득·누설·유출** 을 금지하는 일반 금지 조항이며, §17 (제3자 제공) 와 §18 (목적 외 이용·제공) 는 개인정보의 **합법적 제공·이용** 의 요건을 정한 조항이다 (5과목 08 §3 참조).

| 비교 항목 | §59 (금지행위) | §17·§18 (제공·이용) |
|---|---|---|
| 성격 | 금지 (소극적 의무) | 요건 (적극적 절차) |
| 위반 시 | 형사 처벌 | 형사·행정 처벌 + 손해배상 |
| 적용 대상 | "처리하거나 처리하였던 자" | 개인정보처리자 |

### §6-5. 개인정보취급자의 추가 의무

[개인정보보호법] §28 (개인정보취급자에 대한 감독) [^pipa] 는 개인정보처리자가 개인정보취급자에 대해 다음 의무를 이행할 것을 요구한다.

- 개인정보취급자 범위 최소화 + 명단 관리
- 적절한 관리·감독 (정기 교육·감사)
- 개인정보 처리 권한 설정·제한
- 개인정보 처리 기록 관리

### §6-6. 시험 포인트

- [개인정보보호법] **§59** = 개인정보 취급자의 3대 금지행위.
- §59 의 적용 = **퇴직자 포함** ("처리하였던 자").
- 위반 시 처벌 = **5년 이하 징역 또는 5천만원 이하 벌금** (§71).
- 거짓 동의 취득 = §59 1호 위반.
- 누설 (외부 발설) = §59 2호 위반.
- 권한 초과 변경·유출 = §59 3호 위반.
