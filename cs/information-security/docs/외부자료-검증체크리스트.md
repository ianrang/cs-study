# 정보보안기사(필기) 외부자료 검증 체크리스트

## 1) 원칙
- 1순위: 공식 원문(큐넷/시행기관/법령)
- 2순위: 공공기관 가이드(KISA, 개인정보위 등)
- 3순위: 민간 교재/요약(보조)
- 통합자료(`src/정보보안기사 정리_260214.txt`)는 3순위 보조이며, 1순위와 충돌 시 1순위를 우선한다.

## 2) 필수 확인 소스

### A. 시험체계/과목
- [Q-Net 종목 상세](https://www.q-net.or.kr/crf005.do?id=crf00503&jmCd=1325)
- [KCA 출제기준 게시판](https://www.cq.or.kr/qh_cusgm10_001.do)
- [KCA 정보보안 종목 출제기준 상세](https://www.cq.or.kr/qh_cusgm10_002.do?bbscttNo=65)
- 출제기준 PDF: `docs/info-sec-engineer-criteria-2023-2026.pdf` (적용기간 2023.1.1 ~ 2026.12.31)

### B. 법규(5과목 핵심)
모든 인용은 **국가법령정보센터의 현행본** 기준으로 확인하며, 인용 시 `[법령명 제N조 제M항]` 형식과 시행일자를 명시한다.

- [국가법령정보센터](https://www.law.go.kr/)
  - 개인정보보호법 + 시행령 + 시행규칙
  - 정보통신망 이용촉진 및 정보보호 등에 관한 법률(정통망법) + 시행령
  - 정보통신기반 보호법 + 시행령
  - 전자서명법 + 시행령
  - 정보통신산업진흥법

- [개인정보보호위원회](https://www.pipc.go.kr/) — 표준 개인정보 처리방침, 안전성 확보조치 기준 고시, 가이드라인
- [KISA ISMS-P 인증](https://isms.kisa.or.kr/) — ISMS-P 인증기준(영역·통제항목)

### C. 보안 일반/관리 보완
- [KISA](https://www.kisa.or.kr/) 공개 가이드/보고서
  - SW 개발보안 가이드(시큐어 코딩)
  - 주요정보통신기반시설 기술적·관리적 보호조치 기준
- [OWASP Top 10](https://owasp.org/www-project-top-ten/) — Web 어플리케이션 취약점 분류
- [NIST CSRC](https://csrc.nist.gov/) — SP 800 시리즈, FIPS

### D. 4과목(암호학·인증) 보완
- [KISA 암호이용 활성화](https://seed.kisa.or.kr/) — 국산 암호 알고리즘(SEED, ARIA, LEA, HIGHT, LSH)
- [KCMVP 검증대상 암호모듈](https://seed.kisa.or.kr/kcmvp/) — KCMVP 인증 모듈 목록
- NIST 표준
  - FIPS 197 (AES)
  - FIPS 180-4 (SHA-2)
  - FIPS 202 (SHA-3 / Keccak)
  - FIPS 186-5 (Digital Signature Standard, ECDSA/EdDSA)
  - SP 800-38A/B/C/D (블록 암호 운영 모드)
  - SP 800-56A/B (키 합의)
- [RFC 5246 (TLS 1.2)](https://www.rfc-editor.org/rfc/rfc5246), [RFC 8446 (TLS 1.3)](https://www.rfc-editor.org/rfc/rfc8446)

## 3) 교차검증 절차(실행 순서)
1. 과목명/출제범위는 KCA 출제기준 PDF로 먼저 고정한다.
2. 민간 요약자료 목차를 5과목 체계로 재매핑한다.
3. 법규 항목은 반드시 국가법령정보센터 현행 조문 기준으로 확인하며, **개정 이력을 점검**해 작성 시점의 최신본을 사용한다.
4. 불일치 발견 시 "민간자료 수정"을 우선하며 "공식기준 변경"은 금지한다.
5. **모든 법규 인용은 `[법령명 제N조 제M항]` 형식 + 시행일자 + 원문 링크 각주**로 표기한다.
6. **통합자료(`_260214.txt`)와 출제기준이 충돌하면 출제기준을 우선**한다.
7. **통합자료에 없는 출제기준 세세항목은 외부자료 검증 후 신규 작성**하며 출처를 각주로 단다. 추론으로 채우지 않는다.
8. 누락 방지: 각 과목 README의 "출제기준 매핑표"가 모든 세부항목을 커버하는지 작성 완료 시 확인한다.

- [ ] 5과목(관리및법규) 조문/정의 암기가 진행되고 있는가

- [ ] 인용한 법령 조문이 작성 시점 현행본과 일치하는가
